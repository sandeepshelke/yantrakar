# Yantrakar

Official website for Yantrakar AS.

## Pages

- `index.html` — single-page site (problems, work, about, contact)
- `assets/` — styles

## Run locally

Serve the repository with any static HTTP server, for example:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## Deployment

GitHub Actions deploys the `main` branch to GitHub Pages using `.github/workflows/pages.yml`.
